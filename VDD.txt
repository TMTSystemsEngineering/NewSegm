I. Scope
    The Segmentation Database Software is a Matlab program that generates the
    Segmentation Database, TMT.OPT.TEC.07.044.  The version here generates
    TMT.OPT.TEC.07.044.CCR15.  This is release REL02.

II. References

    This program is under version control on GitHub at:

	https://github.com/TMTSystemsEngineering/NewSegm

IV. Installation & Build Instructions

    Installation is no more than un-zipping the zip file into a directory.
    The software used to build and run in order to create the Segmentation
    Database was:

	MATLAB
	R2015a (8.5.0.197613)
	64-bit (win64)
	February 12, 2015
	License Number: 865865

    The database is generated by running the command:

	NewSegm( false, false )
